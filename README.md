# ESP32 MQTT Soil Moisture Sensor for home assistant
- This Project Is a work in progress and is not any where near finished
## Aim
* the aim of this project is to use ESP32's connected to soil moisture sensors around my house in order to collect data on the moisture levels in plants to add too dashboards, or send notifications when they get too low
